Aquaponics is a sustainable food production system that combines aquaculture (fish farming) and hydroponics (soilless plant cultivation) in a single closed-loop ecosystem. Fish produce natural nutrients that feed the plants, while the plants filter and purify the water, creating a balanced, eco-friendly cycle.

This project enhances traditional aquaponics using sensors, microcontrollers, and automation to enable real-time monitoring and control of critical system parameters, making farming smarter, cleaner, and more efficient.
